Termini di Servizio

Ultimo aggiornamento: 14/03/2026

I presenti Termini di Servizio regolano l'utilizzo del bot Discord privato Luca_Play Bot, disponibile esclusivamente all'interno del server Discord Luca_Play.

1. Ambito di utilizzo

Il bot è un'applicazione privata sviluppata per supportare le attività interne del server Luca_Play. Non è destinato all'uso pubblico generale e può essere utilizzato solo dagli utenti presenti nel server autorizzato.

2. Accettazione dei termini

Utilizzando il bot o interagendo con le sue funzioni all'interno del server Luca_Play, accetti i presenti Termini di Servizio. Se non accetti questi termini, non utilizzare il bot.

3. Uso consentito

È consentito utilizzare il bot solo per gli scopi previsti all'interno del server Luca_Play.

Non è consentito:

  • abusare dei comandi del bot;
  • tentare di aggirare restrizioni, permessi o controlli;
  • utilizzare il bot per attività dannose, spam o molestie;
  • tentare di copiare, compromettere o interrompere il funzionamento del bot.

4. Limitazione dell'accesso

Il proprietario e lo staff autorizzato del server Luca_Play possono limitare, sospendere o revocare in qualsiasi momento l'accesso al bot, a singoli comandi o a specifiche funzioni, senza preavviso, se ritenuto necessario per motivi tecnici, organizzativi o di moderazione.

5. Disponibilità del servizio

Il bot viene fornito "così com'è", senza garanzia di disponibilità continua, assenza di errori o funzionamento ininterrotto. Alcune funzioni possono essere modificate, limitate, sospese o rimosse in qualsiasi momento.

6. Responsabilità

Nei limiti consentiti dalla legge, il proprietario del bot non è responsabile per eventuali malfunzionamenti, interruzioni, perdite di dati, errori nei comandi o danni derivanti dall'utilizzo o dall'impossibilità di utilizzare il bot.

7. Regole del server

L'uso del bot è inoltre soggetto alle regole del server Luca_Play e ai Termini di Servizio di Discord. In caso di violazione delle regole del server, lo staff può intervenire anche sull'accesso al bot.

8. Modifiche ai termini

Questi Termini di Servizio possono essere aggiornati o modificati in qualsiasi momento. La continuazione dell'utilizzo del bot dopo eventuali modifiche costituisce accettazione dei termini aggiornati.

9. Contatti

Per domande, richieste o segnalazioni relative al bot, è possibile contattare lo staff del server Luca_Play tramite il server Discord ufficiale o i canali di supporto interni.

Questo documento si applica esclusivamente al bot privato utilizzato nel server Luca_Play.
